Easily import SEO settings from any XML or CSV file to All In One SEO with the All In One SEO Add-On for WP All Import.

== Description ==

The All In One SEO Add-On for [WP All Import](http://wordpress.org/plugins/wp-all-import "WordPress XML & CSV Import") makes it easy to bulk import your SEO setting to the  All In One SEO plugin in less than 10 minutes.

The left side shows all of the fields that you can import to and the right side displays the data from your XML/CSV file. Then you can simply drag & drop the data from your XML or CSV into the  All In One SEO fields to import them.

The importer is so intuitive it is almost like manually configuring All In One SEO when adding/editing a post.

= Why you should use the All In One SEO Add-On for WP All Import =

* Import new posts, pages, or custom post types and configure their SEO settings all in the same import.

* Update SEO settings for existing posts, pages, or custom post types already published on your site.

* Compatible with all other WP All Import add-ons.

== Installation ==

First, install [WP All Import](http://wordpress.org/plugins/wp-all-import "WordPress XML & CSV Import").

Then install the All In One SEO Add-On.

To install the All In One SEO Add-On, either:

* Upload the plugin from the Plugins page in WordPress

* Unzip import-property-listings-into-realia.zip and upload the contents to /wp-content/plugins/, and then activate the plugin from the Plugins page in WordPress

The All In One SEO Add-On will appear in the Step 3 of WP All Import.
